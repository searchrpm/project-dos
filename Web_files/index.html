UFC Fighter Analysis Approach

The purpose of this project was to get an understanding of fighter profiles from the UFC. We wanted to know which countries produced the most fighters, their age, wins and loss averages and see if there are any insights we can gleam from the data.

Step #1. Gather Data Sources:

Some of our data sources were from Kaggle, Wikipedia and Reddit. Below are some of our main data sources

UFC Wikipedia Page (hundreds of records)
